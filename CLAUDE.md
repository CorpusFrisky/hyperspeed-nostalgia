# Claude Instructions for aiSlopArtHistory

## Output Directory

**ALWAYS save generated images to `examples/` folder, NOT `outputs/`.**

The `examples/` folder is the canonical location for all generated images in this project. Do not create an `outputs/` folder.

## Project Structure

- `src/hyperspeed/` - Main source code
- `examples/` - Generated example images
- `*_GUIDE.md` - Era-specific documentation
